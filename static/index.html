Ce am implementat:

🏭 Depozite

Gestionez depozitele. Pot adauga, modifica si vizualiza depozite.

5 endpoint-uri

🤝 Furnizori

Gestionez furnizorii de la care se cumpara produsele.

5 endpoint-uri

📦 Produse

Gestionez produsele din fiecare depozit. Fiecare produs are un SKU unic.

6 endpoint-uri

📊 Stocuri

Miscari de stoc — adaug, scot sau transfer produse intre depozite.

5 endpoint-uri

Toate endpoint-urile:

Depozite (Warehouses)

POST /api/warehouses/ creaza depozit nou
GET /api/warehouses/ toate depozitele
GET /api/warehouses/{id} un depozit dupa id
PATCH /api/warehouses/{id} modifica partial
PUT /api/warehouses/{id} inlocuieste complet

Furnizori (Suppliers)

POST /api/suppliers/ creaza furnizor nou
GET /api/suppliers/ toti furnizorii
GET /api/suppliers/{id} un furnizor dupa id
PATCH /api/suppliers/{id} modifica partial
PUT /api/suppliers/{id} inlocuieste complet

Produse (Products)

POST /api/warehouses/{id}/products/ adauga produs in depozit
GET /api/warehouses/{id}/products/ produsele dintr-un depozit
GET /api/warehouses/{id}/products/{id} un produs dupa id
PATCH /api/warehouses/{id}/products/{id} modifica partial
PUT /api/warehouses/{id}/products/{id} inlocuieste complet
DELETE /api/warehouses/{id}/products/{id} sterge produs

Stocuri (Inventory)

GET /api/warehouses/{id}/inventory/ stocul dintr-un depozit
GET /api/warehouses/{id}/inventory/{id}/movements miscarile unui produs
POST /api/warehouses/{id}/inventory/{id}/stock-in adauga stoc
POST /api/warehouses/{id}/inventory/{id}/stock-out scoate stoc
POST /api/warehouses/{id}/inventory/{id}/transfer transfera stoc
